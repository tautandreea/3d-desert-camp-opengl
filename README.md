# 3D Desert Camp Scene - OpenGL

## Overview

This project was developed as part of the Computer Graphics course.

The application renders a 3D desert camp scene inspired by the biblical journey of the Israelites through the wilderness. The central element of the scene is the Tabernacle (Tent of Meeting), surrounded by tents, vegetation, animals and environmental objects.
